# fh-txNotify - standalone

<img width="1980" height="1080" alt="fh-txNotify" src="https://github.com/user-attachments/assets/ea20e03b-03f8-44a8-ab5b-7e43ec575fa9" />


Replaces default txAdmin notifications with  NUI.

Warn count is saved per player license.

## Installation

1. Download and put the folder in your `resources` directory
2. Add to `server.cfg`:

```cfg
ensure fh-txNotify
```
3. Edit `config.lua` if you want
4. Restart server
5. Enjoy ;)

## Notify Sound
Replace `ui/notifysound.mp3` with your own file.
